KDPII 검출 결과 비교 (Gold vs k-pii)

KDPII 데이터셋 (Li Fei et al., IEEE Access 2024) 의 30 문서에서 gold 라벨과 k-pii 검출 결과를 시각적으로 비교. 색상으로 표시.

범례:
초록 정탐 = k-pii 가 잡았고 정답에도 있음 (정확히 잡음)
노랑 미탐 = 정답에 있는데 k-pii 가 못 잡음 (놓침)
빨강 오탐 = k-pii 가 잡았는데 정답에 없음 (잘못 잡음)
⚪ 색 없음 = TN (정확히 무시한 일반 텍스트 — 전체의 90%+)
